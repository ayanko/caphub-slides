CapHub

Multiple 'capistrano configuration' in one place.

ayanko.github.com/caphub-slides

Idea

We often use dead-simple mulistage extension

$ cap [STAGE] [RECIPE]

For example

$ cap production deploy
ayanko.github.com/caphub-slides

Benefits

  • Centralized location
  • Reuseable code (DRY)
  • Security policy
